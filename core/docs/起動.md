# Nekodion Backend

> [!NOTE]
> 各設定値は開発メンバーに聞いてください。

## 起動方法

### 準備

1. IntelliJ > 実行 > 構成の編集を開く
2. 新規構成の追加でGradleを選択する

### API

| 項目                | 設定値                                                         |
| ------------------- | -------------------------------------------------------------- |
| 実行 > タスクと引数 | `api:bootRun`                                                  |
| Gradleプロジェクト  | `core`                                                         |
| 環境変数            | `DB_NAME=nekodion;DB_USERNAME=;DB_PASSWORD=;ISSUER=;AUDIENCE=` |
